# AndroidPackageExport
Simple utility that utilizes the Android Debug Bridge to output a list of system packages installed on an android device over USB/WIFI.

# Compatibility:
|         Requirement          |            Value        |
|         ----------           |         ----------      |
|   Minimum Android Version:   |             10          |
|   Minimum Android API:       |             29          |
